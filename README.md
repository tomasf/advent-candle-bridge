# Nordic Advent Candle Bridge

This repository contains the parametric Cadova source code for a classic Nordic-style electric advent candle bridge, available in both 5- and 7-candle versions. You can adjust dimensions, angles, part geometry, and other parameters directly in the code to suit your printer or design preferences. Assembly instructions, photos, and additional details are [available on MakerWorld](https://makerworld.com/en/models/2011951).
